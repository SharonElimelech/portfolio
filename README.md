# Sharon Elimelech — Developer Portfolio

A fast, responsive personal portfolio built with **React**, **TypeScript**, **Vite** and **Tailwind CSS**, showcasing my full-stack and AI-integration work.

**Live:** https://sharon-elimelech.vercel.app

## Tech Stack
- **Frontend:** React, TypeScript, Vite, Tailwind CSS, shadcn/ui
- **Animation:** GSAP (ScrollTrigger)
- **Deployment:** Vercel (CI/CD from GitHub)

## Featured Projects
- **WhatsApp AI Customer-Service Bot** — production AI bot (Node.js, Express, Claude API)
- **Math Tutor App (PWA)** — student & lesson management with offline support
- **CRM Cloud** — client and sales-pipeline tracker
- **Amdocs Job Watcher** — Node.js job-listing automation

## Run Locally
```bash
npm install
npm run dev
```
